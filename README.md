# Indigodomo-DAX88-Actions-Plugin
Some indigodomo scripts I have been working on. 
This plugin works off data provided by FlyingDiver's HomeAssistantAgent to enable easy source selection for up to 8 DAX88 zones.
Download the files, recreate structure and enable the plugin
It will provide an action selection to turn on/off and set source for multiple zones at once.
